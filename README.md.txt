# 🧠 AI SPY — Intelligent Image Caption Generator

AI SPY is an **AI-powered image understanding system** that automatically generates meaningful **captions**, **headlines**, and **social media captions** for any image using computer vision and natural language generation.

---

## 🚀 Overview

Manually writing captions for images is time-consuming and inconsistent.  
AI SPY automates this by combining **image captioning models** (like BLIP) with **language generation models** (like GPT-2) to create human-like, context-aware captions and marketing copy.

---

## 🎯 Features

- 🖼️ **Automatic Image Captioning:** Detects objects and scenes to generate a natural caption.  
- 📰 **Headline Generation:** Creates a short, emotional headline for creative or marketing use.  
- 📱 **Social Caption Generation:** Produces fun and engaging Instagram-style captions with emojis and hashtags.  
- ⚡ **Simple & Lightweight:** Runs locally with CPU support, no heavy GPU needed.
